# Gemini 雑魚敵量産プロンプト

以下をそのまま Gemini に渡してください。

```text
あなたは、インディーゲームのシニア・ナラティブデザイナー兼ピクセルモンスターデザイナーです。

私は「群馬県を異界化したピクセルアートのロードムービーRPG」を作っています。

このゲームの核は以下です。

- 切ない旅
- 土地への敬意
- 理解して通してもらう体験
- シュールさの奥にある、敵と土地の誇りと悲哀

今回ほしいのは、ボスではなく「雑魚敵」の量産案です。
ただし、AIっぽくバラバラなデザインではなく、既存の世界観と実装方式に厳密に合わせてください。

重要な実装ルール:

- 戦闘スプライトは 16x16 のピクセル配列
- 0 は透明
- 1以上は palette 番号
- palette は 4〜6色を基本とする
- 色数を増やしすぎない
- 現在のゲームは、既存敵のシルエット流用 + palette差分 + 10〜20ピクセル程度の改変で統一感を出している
- 毎回ゼロから描くのではなく、「家系」を共有して量産する

敵デザインの基本思想:

- ただのモンスターではない
- その土地の誇りが歪んだ姿
- 1体ごとに「誇り」と「悲哀」が1行で言えること
- 雑魚なので、設定は重すぎない
- ただし、安っぽいネタ敵にはしない

使ってよい家系は以下:

1. daruma系
   - 丸い塊
   - 顔で差分を作る
   - 放置された縁起物、願掛け、目入れの不在に向く

2. guard系
   - 人型の見張り
   - 頭、胴、足が分かれた直立シルエット
   - 関所、見張り、境界、役目に向く

3. thread系
   - 糸、布、機械の影
   - 細身で、手足が少し長い
   - 富岡、工場、止まれない労働に向く

4. mistBeast系
   - 獣と霧の中間
   - 赤城、榛名、湖、霧、迷いに向く

5. mud系
   - 泥、湿原、水底
   - 下半身を崩しやすい
   - 尾瀬、水上、沈んだ記憶に向く

作ってほしい敵は、毎回この手順で設計すること:

1. 土地キーワードを2つ決める
2. 感情を1つに絞る
3. 家系を1つ選ぶ
4. 既存家系との差分を3点だけ決める
5. 16x16 sprite と palette を作る
6. battle.js に貼れる object を出す

必須項目:

- id
- name
- family
- mapTags
- pride
- sorrow
- silhouetteDiff
- hp
- maxHp
- attack
- defense
- goldReward
- expReward
- dropItem
- dropRate
- sprite
- palette

sprite のルール:

- 必ず 16行 x 16列
- 0 は透明
- 情報量を詰め込みすぎない
- 顔、外形、アクセントだけで見分けられるようにする
- 既存ゲームに合う、やや素朴で見やすいドットにする

palette のルール:

- 4〜6色
- 1 = 輪郭 / 最暗部
- 2 = 主色
- 3 = 目や穴などの強い暗部
- 4 = 感情アクセント
- 5,6 = 副素材

数値バランスのルール:

- 雑魚敵なので極端に硬くしない
- 序盤: HP 28〜40
- 中盤: HP 38〜52
- 後半: HP 45〜65
- goldReward と expReward は既存雑魚に近い範囲
- dropRate は 0.12〜0.28 くらい

出力形式:

- JSON のみ
- 1体ずつではなく、配列で出す
- 各 enemy はそのまま JS object に変換しやすい形にする
- 説明文を長々と書かない
- ただし family / pride / sorrow / silhouetteDiff は必ず付ける

silhouetteDiff の出力例:

"silhouetteDiff": [
  "右肩だけ大きく膨らませる",
  "目を1つだけ下げる",
  "足元に泥の尾を追加する"
]

今回作ってほしいセット:

- 地域:
  富岡・下仁田

- 必要数:
  6体

- 内訳:
  - thread系 3体
  - guard系 1体
  - daruma系 1体
  - mud系 or mistBeast系 1体

- 方針:
  - 「産業遺産」「止まれない仕事」「絡まる記憶」「役目を終えたのに働き続ける苦しみ」を感じさせる
  - ただしボスほど重くしない
  - 雑魚らしい分かりやすさを優先する

出力前の自己チェック:

- 16x16 になっているか
- 色数が多すぎないか
- 全員が同じ顔になっていないか
- mapTags と見た目が噛み合っているか
- 既存の世界観から浮いていないか
- battle.js に貼れる粒度になっているか
```

## 使い方

- 地域ごとに `今回作ってほしいセット` だけ差し替える
- 一度に 4〜6体ずつ作る
- 気に入った家系は、次の地域でも流用してよい
- まずは `palette差分だけで成立する敵` を多めに作る
