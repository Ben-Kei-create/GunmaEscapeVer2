# 雑魚敵量産ガイド

このゲームの雑魚敵は、完全に毎回ゼロから描くよりも、
`同じ家系のシルエットを使い回しつつ、役割と土地性だけを差し替える`
ほうが速く、統一感も出ます。

今の実装は [battle.js](/Users/fumiaki/Desktop/GunmaEscapeVer2/js/battle.js) の `enemies` 定義で完結しています。
描画側は [renderer.js](/Users/fumiaki/Desktop/GunmaEscapeVer2/js/renderer.js) の `drawSpriteAbsolute()` が担当しています。

## まず知っておくべき仕様

- スプライトは `16 x 16` の2次元配列。
- `0` は透明。
- `1` 以上はパレット番号。
- 実際の描画色は `palette: { 1:'#xxxxxx', 2:'#xxxxxx' }` で決まる。
- 戦闘画面では通常 `scale: 5` で拡大表示される。
- 雑魚敵は大半が `既存ボス/既存雑魚の sprite を流用して palette だけ変える` 方式でも成立している。

参照例:

- だるま系: [battle.js](/Users/fumiaki/Desktop/GunmaEscapeVer2/js/battle.js#L1156) と [battle.js](/Users/fumiaki/Desktop/GunmaEscapeVer2/js/battle.js#L1241)
- 人型見張り系: [battle.js](/Users/fumiaki/Desktop/GunmaEscapeVer2/js/battle.js#L135) と [battle.js](/Users/fumiaki/Desktop/GunmaEscapeVer2/js/battle.js#L1249)
- 糸/白影系: [battle.js](/Users/fumiaki/Desktop/GunmaEscapeVer2/js/battle.js#L1206) と [battle.js](/Users/fumiaki/Desktop/GunmaEscapeVer2/js/battle.js#L1273)

## 量産の基本ルール

1. 毎回「完全新規」にしない。
2. まず `家系` を決める。
3. 次に `土地の誇り` と `歪み` を1つずつ決める。
4. そのあとで `色`、`顔`、`先端シルエット` だけ変える。

これで、同じ世界の敵に見えつつ、量産しやすくなります。

## 家系の作り方

雑魚敵はまず次の5家系に分けると扱いやすいです。

### 1. だるま系

- 丸い塊
- 正面顔
- 目か眉の欠損で感情を出しやすい
- 少ない修正で大量生産しやすい

向いている要素:

- 縁起物
- 祈願
- 目入れ
- 放置されたご当地土産

### 2. 人型見張り系

- 頭、胴、足が分かれた直立シルエット
- 色替えだけで盗賊、番人、学園関係者、結界守に化ける
- 武器や肩、目の色を変えるだけで差分が出る

向いている要素:

- 関所
- 境界
- 見張り
- 役目に縛られた存在

### 3. 糸/布/機械系

- 細身
- 手足が長く見える
- 白、灰、薄桃、黒で不穏さが出る
- 富岡や工場系と相性が良い

向いている要素:

- 糸
- 織機
- 制服
- ほどけない仕事

### 4. 獣/霧系

- 頭が大きく、胴が短い
- 輪郭を曖昧にすると異界感が出る
- 赤城、榛名、湖、霧で使いやすい

向いている要素:

- 霧
- 風
- 山の獣
- 迷い

### 5. 泥/湿原/水底系

- 下半身を崩す
- 左右非対称を少しだけ入れる
- 泥や水草を輪郭に混ぜる
- 尾瀬や水上に強い

向いている要素:

- 沈んだ記憶
- 泥
- 水底
- 引き留める未練

## 1体を作るときの手順

### 手順1: 土地キーワードを2つ取る

例:

- 富岡なら `生糸` `止まらない機械`
- 草津なら `熱` `近づけない癒やし`
- 尾瀬なら `湿原` `沈んだ記憶`

### 手順2: 感情を1つに絞る

1体につき感情は1つで十分です。

候補:

- 空虚
- 役目への執着
- 孤独
- 未練
- 痛み
- 誇り

### 手順3: 家系を1つ選ぶ

例:

- `放置された土産` なら だるま系
- `止まれない工員の影` なら 糸/機械系
- `霧に混じる湖畔の幼獣` なら 獣/霧系

### 手順4: 見た目の差分を3点だけ入れる

大量生産では、差分は3点だけで十分です。

- 顔: 目の位置、数、欠け
- 外形: 頭の突起、肩、尻尾、裾
- アクセント: 模様、傷、湯気、泥、糸

### 手順5: パレットを4〜6色に制限する

おすすめは次の役割分担です。

- `1`: いちばん暗い輪郭
- `2`: 体の主色
- `3`: 目や穴などの最暗部
- `4`: 頬、口、印、傷などの感情アクセント
- `5`: 服、糸、湯気、泥などの副素材
- `6`: 足や腕などの補助色

7色以上に増やすと、今のゲームの戦闘画面では情報量がうるさくなりやすいです。

## Gemini 量産向けのコツ

Gemini に毎回「敵を考えて」と投げると、設定だけ増えて絵が統一されません。
代わりに、以下の順で固定すると安定します。

1. 家系を固定する
2. 16x16 配列で出させる
3. パレット数を制限する
4. `既存敵との差分3点` を先に言わせる
5. 最後に `battle.js に貼れる object` 形式で出させる

## 量産時のおすすめ比率

20体一気に作るなら、次の比率が安全です。

- `12体`: 既存 sprite 流用 + palette 差分
- `6体`: 既存 sprite を 10〜20 ピクセル修正
- `2体`: 完全新規シルエット

これで世界観が崩れず、作業量も暴走しにくいです。

## やってはいけないこと

- 32x32 以上の高密度スプライトを急に混ぜる
- 1体だけ色数を増やしすぎる
- 輪郭線なしの“ふわっとしたAI絵”に寄せる
- ただの強さ違いで、土地性がない敵を増やす
- 似た敵を増やすのに、毎回ゼロから考える

## 量産時の判定基準

新しい雑魚敵は、次の5つを満たせば通してよいです。

1. `どの土地の敵か` が一目で分かる
2. `家系` が既存敵と繋がっている
3. `誇り` と `悲哀` が1行で言える
4. `16x16 / 4〜6色` に収まっている
5. 既存2体と並べても、顔か輪郭か色のどれかで見分けがつく

## 実務上いちばん効く運用

Gemini には、1回で30体作らせるより、
`1地域につき3体ずつ`
作らせるほうが質が安定します。

おすすめの束ね方:

- 前橋・高崎セット
- 富岡・下仁田セット
- 草津・白根セット
- 赤城・榛名セット
- 水上・尾瀬セット

この単位で作ると、同じ地方の中で「似ているけど違う」敵を揃えやすいです。
