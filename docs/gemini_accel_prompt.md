# Gemini 実装加速プロンプト

以下をそのまま Gemini に渡してください。

```text
あなたはインディーゲームのシニア・ナラティブデザイナー兼レベルデザイナーです。
私は「群馬県を異界化したピクセルアートのロードムービーRPG」を作っています。

このゲームの新しい企画方針は以下です。

- 一文で言うと: 「群馬の誇りと悲哀を辿る、ピクセルアートの異界ロードムービーRPG」
- 感情の核: 切ない旅 / 土地への敬意
- プレイ時間: 5〜8時間
- 構造: 6章 + 終章
- 戦闘: ダイスを使った「リスペクト前提の儀式」。物語6 : 戦闘4
- 目標: 最初は群馬ネタの変なゲームに見えるが、最後には静かに胸に残る作品にする
- エンディングの余韻: 静かに沁みる、少しの希望

重要な感情ピークは3つです。

1. 序盤の転調
   高崎・前橋で「変なご当地ネタ」から「土地の誇りを守る悲哀」へ反転する

2. 中盤のカタルシス
   草津・白根・水の異界で、「倒す」のではなく「誇りを満たして浄化する」へルールが変わる

3. 終盤の真相
   谷川・尾瀬・水上で、脱出と記憶の真相が交差し、これまで学んだ儀式知識を総動員する

戦闘のアハ体験は3回に絞ります。

1. 基礎: ダイスとアイテムを組み合わせる
2. 反転: あえて悪い目を出して相手を誘導する
3. 結合: 探索用アイテムが最終儀式の触媒だったと気づく

今回あなたにやってほしいのは、コードではなく「実装に直結する設計素材」の作成です。

以下を出してください。

1. 6章+終章の詳細章立て
   - 各章のタイトル
   - その章のランドマーク
   - その章の感情
   - その章で学ぶ儀式ルール
   - その章のボスの誇りと悲哀

2. 各章で「環境だけで語る」演出案を3つずつ
   - テキスト説明ではなく、看板、地形、残骸、音、遠景で語ること

3. 3つの感情ピークのシーン詳細
   - 画面の見た目
   - BGM/環境音
   - プレイヤーがその瞬間に理解すること
   - その直後に起こる行動

4. Steamの看板画像候補を3案
   - すべて「一枚で切ない異界ロードムービー」と伝わること
   - UIは極力映さないこと

5. 最後に、「このゲームで絶対に削ってはいけない要素」と「削ってよい要素」を表で整理してください

出力ルール:
- 一般論は少なめに
- すぐ制作に使える具体案を優先
- 箇条書き中心でよい
- 日本語で
- 1章ごとに見出しをつける

このプロジェクトのソース・オブ・トゥルースは、以下の仕様です。
「6章+終章」「切ない旅」「土地への敬意」「知識で解く儀式」

もし迷ったら、必ず「プレイヤーに何を感じさせたいか」を優先してください。
```
