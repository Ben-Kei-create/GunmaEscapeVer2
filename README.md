# 群馬県からの脱出 / Escape from Gunma

群馬県を異界化したピクセルアートRPGです。  
現在は「群馬の誇りと悲哀を辿る、ピクセルアートの異界ロードムービーRPG」という方針で再設計と実装を進めています。

## 現在のコンセプト

- シュールなご当地ネタだけで終わらせず、`土地への敬意` と `切ない旅` を主軸にする
- プレイヤーは群馬の深部へ進み続けながら、歪んだ土地や敵の誇りと悲哀に触れていく
- 戦闘は単なる撃破ではなく、章によっては「理解して通してもらう」儀式へ反転する

## 現在の実装状況

- タイトル画面
  - `はじめから / つづきから / あいことば / 実績`
- セーブ
  - `localStorage` による通常セーブ
  - PC持ち運び用の `あいことば`
- オープニング
  - `はじめから` で即探索開始せず、ミニムービーから導入
- 旅路UI
  - 6章+終章ベースの旅路トラッカー
  - 章タイトル、現在目標、エリアバナー
- 通常探索
  - マップ移動、会話、買い物、イベント進行
- 通常戦闘
  - ダイス戦闘
  - 雑魚敵との通常遭遇
  - 複数敵編成に対応
- 儀式戦闘
  - `repair_eye`
  - `untangle`
  - `temperature`
- 仲間
  - 最大3人まで同行
  - 現在は `支援ボーナス + UI表示` を実装
- 状態管理
  - `respectGauge`
  - `catalysts`
  - `partyMembers`

## 操作

- `矢印キー / WASD`: 移動
- `Z / Enter / Space`: 決定
- `X / Esc`: キャンセル、メニュー
- `M`: ミュート
- 戦闘中 `← / →`: 複数敵の対象切替

## ローカルでの起動

ビルドは不要です。  
ルートディレクトリで静的サーバーを立てて、ブラウザで開いてください。

```bash
python3 -m http.server 4173
```

ブラウザで以下を開きます。

```text
http://127.0.0.1:4173
```

## デバッグ用の補助

- `window.render_game_to_text()`
  - 現在のゲーム状態を JSON で返します
- `window.advanceTime(ms)`
  - テスト用の時間前進フックです
- `?debugBattle=threadMaiden`
  - 富岡の儀式戦デバッグ
- `?debugBattle=yubatake_guardian`
  - 草津の儀式戦デバッグ
